# 2026-07-07 SoC 官方回归交接

## 当前结论

- 自编 SoC 指令级回归已通过：`tb_soc_custom_instr` 为 39/39 PASS。
- 官方 HEX harness 已可稳定驱动 `soc_top`，不再停在早期 `rv32ui-p-simple` 启动片段。
- 官方支持集通过：
  - RV32UI：41/41 PASS。
  - RV32UM：8/8 PASS，覆盖 `div/divu/mul/mulh/mulhsu/mulhu/rem/remu`。
  - RV32MI CSR：`rv32mi-p-csr` PASS，`rv32mi-p-mcsr` PASS。
- `rv32ui-p-fence_i` 暂不纳入支持集，因为该官方 case 会在运行时改 IMem；当前最终 SoC
  规则明确 instruction memory 不允许运行时修改。

## 本轮关键修订

- `csr_file` 增加只读 `misa` 和 machine ID CSR 支持：
  - `misa = 32'h4000_1100`，对应 RV32IM。
  - `mvendorid/marchid/mimpid` 返回 0。
  - 对 `misa/mvendorid/marchid/mimpid/mhartid` 的写入会产生 illegal CSR。
- `tb_csr_file` 增加 `misa` 和 machine ID CSR directed 检查。
- 官方 HEX 测试台保留 `gp == 1` 判定规则，并继续用同一 HEX 初始化 IMem/DMem。
- SoC 自编测试和官方测试均不依赖官方启动外的额外 pass/fail 约定。
